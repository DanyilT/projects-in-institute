### Skills Demo Four (8/10%)

Generate Java (or suitable Object‐ Oriented language) classes from UML class diagram specifications
```text
Using the UML class diagram provided create relevant code in JavaScript. This code should be commented and well documented. Using correct use of classes, methods, functions etc. Outline your understanding of the UML class in a word document discussing your choices and reasons behind your coding methodology.

- Appropriate implementation of code (3)
- Class Implemented correctly (2)
- Methods correctly implemented (2)
- Well Commented code (2)
```

Zoo management system demonstrating:
- Outer [`Zoo`](Skills%20Demo%20Four/zoo.java) class managing animals and enclosures
- Inner `Enclosure` class for grouping animals by food type
- Static nested `Gate` class for entrance management
- Lists to store enclosures and animals
- Methods for adding enclosures, counting animals, and listing information

**Features:**
- Private instance variables with proper encapsulation
- Constructor initialization
- List management using ArrayList
- Inner class access to outer class properties
- toString method implementation for display
