### Skills Demo Four

Zoo management system demonstrating:
- Outer [`Zoo`](Skills%20Demo%20Four/zoo.java) class managing animals and enclosures
- Inner `Enclosure` class for grouping animals by food type
- Static nested `Gate` class for entrance management
- Lists to store enclosures and animals
- Methods for adding enclosures, counting animals, and listing information

**Features:**
- Private instance variables with proper encapsulation
- Constructor initialization
- List management using ArrayList
- Inner class access to outer class properties
- toString method implementation for display

**Files:**
- [zoo.java](Skills%20Demo%20Four/zoo.java) - Complete zoo management system
- [Documentation for zoo.pdf](Skills%20Demo%20Four/Documentation%20for%20zoo.pdf) - Technical documentation
