# Programming & Design Principles

This folder contains Java projects and exercises for programming and design principles.

## Projects

- **calculator_CAO**: CAO points calculator with login system
- **quiz**: Console-based quiz game with user registration and scoring
- **exam**: Java exam's exercises

### calculator_CAO

- [calculator_CAO](src/calculator_CAO) - A Java project for calculating CAO points with a login system.

A comprehensive CAO points calculator that includes:
- User registration and login system ([logUp.java](src/calculator_CAO/logUp.java), [logIn.java](src/calculator_CAO/logIn.java))
- Points calculation for multiple subjects ([Calculator.java](src/calculator_CAO/Calculator.java))
- Pass/Merit/Distinction grade evaluation
- Main application entry point ([Main.java](src/calculator_CAO/Main.java))

**Files:**
- [Main.java](src/calculator_CAO/Main.java) - Application entry point
- [Calculator.java](src/calculator_CAO/Calculator.java) - CAO points calculation logic
- [logUp.java](src/calculator_CAO/logUp.java) - User registration
- [logIn.java](src/calculator_CAO/logIn.java) - User authentication

### quiz

- [quiz](src/quiz) - A console-based quiz game with user registration and scoring.

A console-based quiz game with the following features:
- User registration and login ([Start.java](src/quiz/Start.java))
- Multiple-choice questions with various topics ([DATA.java](src/quiz/DATA.java))
- Score tracking and feedback ([Questioner.java](src/quiz/Questioner.java))
- Cheat code functionality ([Process.java](src/quiz/Process.java))
- Replay option

**Files:**
- [Main.java](src/quiz/Main.java) - Application entry point
- [Start.java](src/quiz/Start.java) - Welcome screen and authentication
- [Process.java](src/quiz/Process.java) - Game flow control
- [Questioner.java](src/quiz/Questioner.java) - Question presentation and scoring
- [DATA.java](src/quiz/DATA.java) - Question and answer database

## exam

- [exam](src/exam) - Java exam's exercises.

- [IntegerSum.java](src/exam/IntegerSum.java) - Prompts user for three integers and calculates their sum
- [CheckLargest.java](src/exam/CheckLargest.java) - Compares two numbers and determines which is larger
- [FortuneTeller.java](src/exam/FortuneTeller.java) - Fortune telling app using switch statements
- [EvenNumbers.java](src/exam/EvenNumbers.java) - Displays even numbers from 2 to 100 using a for loop
