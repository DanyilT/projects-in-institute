# Multimedia Authoring

This folder contains projects and resources for multimedia authoring, focusing on Unity 3D and Augmented Reality.

## Projects

- [**ChillZone**](#chillzone): Android Augmented Reality (AR) game APK and Unity project.

### Assessment Brief

<details>

```text
Each student will create a APK android application that will have “augmented reality” component. This application will have a splash screen, menu and interactive elements. Students will create software that is logical and navigable and that has a clear “objective” or “purpose”. 
Students will create drafts of this software and test these drafts. The students will gather feedback on this testing and will integrate the feedback into future versions of the software.
```

### ChillZone

An Augmented Reality game where players throw a ball into a bin to score points.

**Gameplay:**
1. Scan the surface with your device to detect optimal placement area
2. Tap on detected surface to place the bin in AR environment
3. Use AR Toggle button to show/hide augmented reality elements
4. Throw the ball into the bin to score
5. Use Restart button to spawn a new ball after a miss
6. Check your best score with the Score button

**Tips:**
- Ensure adequate lighting for optimal surface detection
- Use a flat surface for best results
- Experiment with different angles and distances
- Challenge yourself by increasing difficulty
