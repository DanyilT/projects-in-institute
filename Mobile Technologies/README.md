### Coffee Shop Website - EspressoEmpire

- [coffeeShop](src/coffeeShop) - A responsive coffee shop website with product ordering.

A fully responsive coffee shop website featuring:

**Pages:**
- [index.html](src/coffeeShop/pages/index.html) - Home page with carousel and welcome section
- [assortiment.html](src/coffeeShop/pages/assortiment.html) - Product catalog with 6 coffee varieties

**Features:**
- Interactive image carousel with automatic rotation
- Product modal windows with customizable options:
  - 95 different milk types selection
  - Size options (125ml, 250ml, 500ml)
  - Portion quantity selector
  - Dynamic price calculation
- User account system with login/signup forms
- Google Maps location integration
- Support chat interface
- Responsive navigation with mobile menu toggle

**Technologies:**
- HTML5 semantic markup
- CSS3 with custom properties and responsive grid
- Vanilla JavaScript with ES6+ features
- Font Awesome icons
- Mobile-first responsive design

**Files:**
- [index.html](src/coffeeShop/pages/index.html) - Home page
- [assortiment.html](src/coffeeShop/pages/assortiment.html) - Products page
- [script.js](src/coffeeShop/pages/scripts/script.js) - Interactive functionality
- [style.css](src/coffeeShop/pages/styles/style.css) - Main styles
- [headbar-footer.css](src/coffeeShop/pages/styles/headbar-footer.css) - Navigation and footer styles
